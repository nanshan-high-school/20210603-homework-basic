# 20210603 購物網站 struct 版

1. 使用 struct 設定多種產品（至少三種）以及其金額和庫存數
2. 撰寫一個購物車程式，消費者可以指定購買的產品以及數量
3. 輸入優惠代碼可以折抵金額（「滿 100 折 50」跟「滿一百八折」兩種優惠代碼）
4. 如果總金額低於 490，就要付運費 60 元。如果優惠代碼套用後低於 490 仍須付運費 60 元
5. 計算最後總金額及顯示購買明細
